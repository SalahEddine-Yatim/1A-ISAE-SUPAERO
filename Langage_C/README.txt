Welcome to your private repository for the TCS1-IN lecture at SUPAERO!

The material for your lab session will be distributed through this
Subversion repository. You will also have to submit your project using
this repository.

Do not forget to update your working copy at the beginning of each
session with the following command:

svn update

To create a local working copy of this repository, use the following
command:

svn checkout https://eduforge.isae.fr/repos/IN101/5B/LOGIN LOCAL_DIR

where LOGIN is your login and LOCAL_DIR the directory in which you
want to create the working copy.
